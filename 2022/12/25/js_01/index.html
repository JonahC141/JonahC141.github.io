JavaScript常用方法

  • 📒 js基础
  • 🏷️ JavaScript

Number类型方法

Number.parseFloat()

此方法用于获取一个数字的整数和浮点数部分，也可以将一个非数字类型转化为数字类型

Number.parseInt()

此方法跟parseFloat很相似，区别在于，parseInt取数字的整数部分

Number.toString()

此方法用于将数字转化为字符串

以下string方法不改变原字符串

concat（）

​ 用于拼接字符串
​ A.concat(B)

a.length

用于获取字符串的长度

a.indexof()

用于返回字符串中某一个字符的位置，从左往右查找，
从0开始，类似于数组

a.lastindexof()

用于返回字符串中某一个字符的位置，从右往左查找

replace()

方法用于替换某一段字符，返回一个新的字符串
a.replace(‘abc’,’123’)

toUpperCase()

此方法用于将字符串中的小写字母转化为大写字母

toLowerCase()

此方法用于将字符串中的大写字母转化为小写字母

split()

此方法用于将一个字符串以特定的格式切割成数组
可以在方法调用后使用 [i]访问数组中的某一段字符
var str = ‘a,b,c,d,efg’;
str.split(‘,’)[2]

slice()

用于切割字符串中某一段，不改变原字符串，传入两个参数，
第一个是切割的起始位置，第二个是切割的结束为止，左闭右开区间
var str = ‘a,b,c,d,efg’;
console.log(str.slice(2,5));
字符串中的“，”也是元素之一，不可忽略

以下是array方法

Array.isArray()

用于判断是否为一个数组，返回布尔值

array.of()

用于将传入参数组成数组

array.from()

此方法将一个数组对象转化为一个数组，不改变原数组，

    var arr = [1,2,3,4,5,1,2];
    var set = new Set(arr);  //Set(5) { 1, 2, 3, 4, 5 }
    var arr2 = Array.from(set);
    console.log(arr2); //[ 1, 2, 3, 4, 5 ]
pop()

此方法删除数组最后一个元素，改变原数组，返回被删除的元素，

shift()

此方法删除数组第一个元素，改变原数组，返回被删除元素

push()

此方法向数组末尾添加一个元素，改变原数组，返回添加后数组的 length

unshift()

此方法向数组开头添加一个元素，改变原数组，返回添加后数组的 length

reverse()

反转并且返回原数组

splice()

此方法接受3个参数，

​ 第一个参数（必填），选中数组中某一个位置，可以为复数（负数从最后一项开始算起）
​ 第二个参数（选填），规定要 [删除] 的长度，可以为0
​ 第三个参数（选填），规定插入的元素
​ 第四(n)个参数（选填），规定插入的元素

上述代码中第一个参数选中数组第0项的位置，第二个参数设定截取长度为2，调用此方法返回被截取的数组，同时改变原数组。

sort()

将数组进行排序，会改变原数组，非必填的参数，是一个函数

b-a负数升序，正数降序

copyWithin()

在此方法中接收了三个参数，先介绍一下三个参数的用途：

第一个参数：代表要粘贴的起始位置

第二个参数：复制的起始位置

第三个参数：复制的截至位置(不包含截至位置的值)即 左闭右开 区间

fill()

覆盖数组

slice()

复制数组中的某一部分元素，不改变原数组

两个参数，开始和结束，左闭右开区间

join()

根据所传递的参数连接数组中的每一个元素，返回连接好的字符串

不传参数，默认用逗号连接，传递参数为任意值，则以任意值连接不改变原数组

concat()

将数组与某一个值连接，也可以连接一个或多个数组，不改变原数组

toString()

将一位数组或者多维数组转化为字符串，不改变原数组

indexOf()

此方法用来查找数组中某一项的下标值，从左往右查找，不改变原数组

lastIndexOf()

此方法也是用来查找数组中某一项的下标值，从右往左查找，不改变原数组

includes()

此方法用来查找数组中某一项是否存在，返回true或者false

遍历数组，不改变原数组

forEach()

此方法用来遍历数组，不能改变原数组

第一个参数(必选)：便利的每一项的值

第二个参数(可选)：遍历的每一项的下标值

第三个参数(可选)：被遍历的数组

第四个参数(可选)：执行回调函数时的this

some()

遍历数组，看是否有符合条件的值

every()

此方法用来检测数组中的元素是否全部满足条件

filter()

此方法用来筛选数组中的元素，符合筛选条件的元素会被返回，

组成新的数组，不改变原数组

map()

此方法用于处理数组中的每一个元素，并且返回新数组

reduce()

数组累加器

第一个参数：per，如果规定了第五个参数，则per为第五个参数的值，如果没有规定第五个参数，则为数组中第一项

第二个参数：item，如果per为数组第一项，则item为数组第二项，如果per为规定的第五个参数值，则item为数组第一项

第三个参数：item的索引值

第四个参数：原数组

第五个参数：初始值

flat()

此方法会深度遍历并且展开数组，接收可选参数，可选参数的值为区间1到无穷。常用于展开多维数组

Math

Math.random()

用于产生0-1之间的随机数，可以自定义随机数区间

获取0-100的随机数

Math.max()

Math.max可以接受一组数字，返回这组数中最大的数

Math.min()

Math.min可以接受一组数字，返回这组数中的最小的数

圆周率使用 Math.PI
